# Modules

This directory contains the core runtime implementation for Glitch Cricket Engine.

## Major Areas

- state normalization
- prediction and scenario logic
- context and contradiction handling
- provider clients
- recorder / review infrastructure
- Telegram and output helpers

## Notable Files

- `match_state.py`
- `innings_state.py`
- `predictor.py`
- `scenario_model.py`
- `wicket_hazard.py`
- `chase_state.py`
- `match_context.py`
- `match_recorder.py`

If you are new to the codebase, start with `match_state.py`, `innings_state.py`, and `predictor.py` first.
